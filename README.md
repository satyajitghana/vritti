**450+ Components** · **160+ Blocks** · **9 Categories** · **42 Themes**

A collection of beautifully designed React components, page blocks, and a visual theme editor. Built with React 19, Tailwind CSS v4, and Motion. Open source. Copy and paste.

## Features

- **Components** — Animations, backgrounds, text effects, buttons, layouts, inputs, media players, and more
- **Blocks** — Hero sections, pricing tables, auth forms, testimonials, dashboards, and modals
- **Theme Editor** — 42 presets, Google Fonts, contrast checker, CSS export
- **AI Components** — Chat streaming, image generation, voice, AI search
- **Interactive Playground** — Configure component props in real-time
- **shadcn CLI** — Install with `npx shadcn@latest add @vritti/component-name`

## Tech Stack

React 19 · Next.js 16 · Tailwind CSS v4 · Motion · TypeScript · Fumadocs
