# Introduction

Vritti UI is a modern React component library designed for developers who want to build beautiful, performant web applications quickly.

## What is Vritti UI?

Vritti UI provides a comprehensive collection of pre-built, customizable React components that you can copy and paste directly into your projects. Unlike traditional component libraries, there's no npm package to installâ€”just copy the code you need.

## Key Features

### 184+ Components
From simple buttons to complex 3D effects, we have components for every use case.

### Modern Tech Stack
Built with the latest web technologies:
- Next.js 16
- React 19
- Tailwind CSS v4
- Motion (Framer Motion)
- Three.js for 3D components

### No Installation Required
Copy and paste components directly into your project. Full control, zero dependencies (except for the component's listed dependencies).

### Fully Typed
Every component is built with TypeScript for excellent developer experience and type safety.

### Customizable
Built with Tailwind CSS, making it easy to customize components to match your design system.

## Philosophy

Vritti UI believes in:
- **Transparency**: You own the code
- **Flexibility**: Customize everything
- **Performance**: Optimized for speed
- **Beauty**: Design matters
