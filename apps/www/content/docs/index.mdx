# Welcome to Vritti UI

Vritti UI is a collection of 184+ beautiful React components built with Next.js 16, Tailwind CSS v4, and Motion animations.

## Quick Start

Get started by installing Vritti UI components in your project.

```bash
npx shadcn add [component-name]
```

## Features

- **184+ Components**: Animations, backgrounds, text effects, and more
- **Copy & Paste**: No npm install required
- **Modern Stack**: Built with Next.js 16, Tailwind CSS v4, Motion
- **TypeScript**: Fully typed for better developer experience
- **Customizable**: Easy to modify with Tailwind CSS

## Categories

Explore our components organized into 7 categories:

- **Backgrounds**: Background patterns and effects
- **Animations**: Motion and animated components
- **Text Effects**: Text animations and effects
- **Buttons**: Interactive button components
- **Layouts**: Layout and grid systems
- **Shaders**: Advanced shader-based effects
- **Special**: Complex and unique components
