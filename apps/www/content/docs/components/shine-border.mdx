## Props

### Shine Border

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `borderWidth` | `number` | `1` | Width of the border in pixels |
| `duration` | `number` | `14` | Duration of the shine animation in seconds |
| `shineColor` | `string \| string[]` | `"#000000"` | Color of the shine effect, can be a single color or an array of colors |
| `className` | `string` | `-` | Additional CSS classes |
| `style` | `React.CSSProperties` | `-` | Custom inline styles |
