## Props

### Orbiting Circles

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `className` | `string` | `-` | Additional CSS classes for each orbiting item |
| `children` | `React.ReactNode` | `-` | Elements to place in orbit |
| `reverse` | `boolean` | `-` | Whether to reverse the orbit direction |
| `duration` | `number` | `20` | Base duration of one full orbit in seconds |
| `radius` | `number` | `160` | Radius of the orbital path in pixels |
| `path` | `boolean` | `true` | Whether to render the circular orbit path |
| `iconSize` | `number` | `30` | Size of each orbiting icon container in pixels |
| `speed` | `number` | `1` | Speed multiplier for the orbit animation |
