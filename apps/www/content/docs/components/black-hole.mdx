## Features

- **Accretion Disc**: Animated with Perlin 3D noise for realistic texture
- **Particle System**: Up to 50,000 orbiting particles with parametric orbital mechanics
- **Gravitational Lensing**: Screen-space distortion effect around the black hole
- **Chromatic Aberration**: RGB color split at 120Â° intervals for light bending effect
- **Three-Pass Rendering**: High-quality multi-pass rendering pipeline
- **Quality Presets**: Low (10k particles), Medium (25k), High (50k)

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `innerColor` | `string` | `"#ff8080"` | Inner color of the accretion disc (closest to black hole) |
| `outerColor` | `string` | `"#3633ff"` | Outer color of the accretion disc (edge) |
| `quality` | `"low" \| "medium" \| "high"` | `"medium"` | Quality preset affecting particle count and render resolution |
| `particleCount` | `number` | Quality-based | Number of orbiting particles (overrides quality preset) |
| `particleSize` | `number` | `0.015` | Size multiplier for particles |
| `rotationSpeed` | `number` | `0.3` | Speed multiplier for disc rotation |
| `orbitSpeed` | `number` | `1.0` | Speed multiplier for particle orbital motion |
| `disableAnimation` | `boolean` | `false` | Disable all animation (static render) |
| `rgbShiftRadius` | `number` | `0.00001` | Intensity of chromatic aberration (RGB color split) |
| `distortionStrength` | `number` | `1.0` | Strength of gravitational lensing distortion effect |
| `className` | `string` | `""` | Additional CSS classes for the container |

## Technical Details

The black hole effect uses a three-pass rendering pipeline:

1. **Space Scene Pass**: Renders the accretion disc and particle system to a high-resolution texture
2. **Distortion Mask Pass**: Renders a radial distortion field to a low-resolution texture
3. **Final Composition Pass**: Combines both textures with gravitational lensing distortion and chromatic aberration

The component uses OGL (Open Graphics Library) for direct WebGL control, providing excellent performance and visual quality. Based on Bruno Simon's webgl-black-hole reference implementation.
